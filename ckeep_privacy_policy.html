Privacy Policy

Effective Date: 26.03.2025

1. General Provisions

The Ckeep application (hereinafter referred to as the "Application") is designed for the secure storage of user passwords. We place a strong emphasis on privacy and data security.

2. Data Collection and Use

The Application does not collect, transmit, or process users' personal data. All information is stored exclusively on the user's device.

3. Data Storage

  • All data entered by the user into the Application is stored locally on the device.
  • The data is encrypted and protected by a password that the user enters when logging into the Application.
  • The Application does not transmit, synchronize, or upload data to servers or cloud storage.

4. Data Security

  • The encryption key is known only to the user and is not stored within the Application.
  • We do not have access to user data and cannot restore it in case of key loss.

5. Data Sharing with Third Parties

The Application does not transfer or disclose information to third parties. All data remains on the user's device.

6. User Responsibility

  • The user is solely responsible for keeping their access key secure.
  • In case of key loss, the data cannot be recovered.

7. Changes to the Privacy Policy

We reserve the right to make changes to this Privacy Policy.
